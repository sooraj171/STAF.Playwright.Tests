# QA orchestrator artifacts

This folder holds **markdown outputs** from the **QA Orchestrator** workflow (work-item–driven STLC).

- **Reports:** `work-items/{Provider}-{WorkItemId}/` (see `AI/instructions/qa-orchestrator-lifecycle.md`).
- **Source of truth for the process:** `AI/instructions/qa-orchestrator-lifecycle.md` and `AI/skills/qa-orchestrator.md`.

Commit these reports if your team wants history in git; otherwise add `work-items/` to `.gitignore`.
